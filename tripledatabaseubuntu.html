Website Wordpress di Ubuntu server dengan Database yang terpisah

Topologi

Web Server

Step 1. Menginstall Apache2 pada Web Server

Apache tersedia dalam repositori perangkat lunak default Ubuntu, sehingga memungkinkan untuk menginstalnya menggunakan tools package management konvensional.

Kita mulai dengan mengupdate package index server kita dengan:

$ sudo apt update

Dilanjutkan dengan menggunakan apt install untuk menginstall Apache2:

$ sudo apt install apache2

Konfirmasi instalasi dengan ketik ‘y’ lalu enter, apt akan lanjut menginstall apache dan keperluan yang dibutuhkan.

Untuk konfirmasi apakah apache2 berhasil diinstal pada server pakai perintah:

$ sudo systemctl status apache2

Seperti yang dikonfirmasi oleh output ini, dimana di baris Active dikembalikan ‘active (running)’ berarti layanan telah dimulai dengan sukses. Namun, cara terbaik untuk menguji ini adalah dengan membuka halaman default arahan dari Apache.

Menggunakan IP Address di network adapter bridged adapter:

Di Virtual box, arahkan mouse ke Devices -> Network lalu klik Network Settings.

Ganti di isi dari ‘Attached to :’ menjadi bridged adapter dengan mengklik dan memilih bridge adapter.

Kita dapat mengakses halaman arahan Apache default melalui alamat IP. Jika tidak tahu alamat IP server, kita bisa mendapatkannya dengan beberapa cara berbeda dari baris perintah.

$ ip a

Alamat IP akan dapat ditemukan di baris inet nomor 2. Atau bisa juga dengan:

$ hostname –I

Akan dikembalikan beberapa alamat yang dipisahkan oleh spasi. Kita dapat mencoba masing-masing di browser web untuk menentukan apakah mereka berfungsi.

Saat anda telah menemukan IP address anda, masukkan IP tersebut ke browser anda:

http://server_ip/

Selamat, apache2 berhasil diinstall di web server!

Step 2. Menginstall mysql-client pada Web Server

Program yang berinteraksi dengan server MySQL dikenal sebagai MySQL Client. Command-line toolsnya memungkinkan kita untuk berinteraksi dengan server dan juga untuk menjalankan perintah di shell scripts atau program lain.

Untuk menginstall mysql-client, jalankan perintah:

$ sudo apt install mysql-client

Klik y dan enter, untuk melanjutkan instalasi.

Step 3. Menginstall PHP pada Web Server

Untuk menginstall package php dan php-mysql, jalankan:

$ sudo apt php php-mysql

Saat diminta, ketik Y dan ENTER untuk mengonfirmasi pemasangan.

Untuk mengonfirmasi php telah terinstall, buat file info.php ke /var/www/html/ dengan perintah:

$ sudo nano /var/www/html/info.php

Kemudian masukkan teks yang seperti kedalam file tersebut:

<?php

phpinfo();

?>

Untuk menyimpan, klik CTRL + O lalu ENTER dan keluar dengan CTRL + X.

Kunjungi URL ip address server anda diikuti dengan /info.php:

http://ip_address/info.php

PHP berhasil diinstall di server! Kita akan melihat halaman web yang berisi informasi rinci tentang servernya. Setelah memeriksa informasi yang relevan tentang server PHP melalui halaman itu, baik untuk menghapus info.php tadi karena berisi informasi sensitif tentang lingkungan PHP dan server Ubuntu yang dipakai. Kita dapat menggunakan rm untuk menghapus file itu:

$ sudo rm /var/www/html/info.php

Database Server

Step 1. Instalasi mysql-server di Database Server

Kita mulai dengan mengupdate package index server kita dengan:

$ sudo apt update

Lalu install package mysql-server:

$ sudo apt install mysql-server

Saat diminta klik Y dan ENTER.

Step 2. Mengedit konfigurasi di mysql

Kita dapat mengubah konfigurasi mysql server di file mysqld.cnf yang berada di direktori /etc/mysql/mysql.conf.d/. Kita akan membuka file ini dengan (sebelumnya siapkan alamat ip server database) :

$ sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf

Temukan parameter bind-address, ini memberi tahu perangkat lunak basis data di alamat jaringan mana untuk mendengarkan koneksi. Ubah nilai parameter bind-address menjadi ip address database server.

Simpan dan tutup dengan menekan CTRL + X, Y, lalu ENTER.

Untuk mengupdate konfigurasi, restart databasenya:

$ sudo systemctl restart mysql

Step 3. Membuat user untuk web server agar dapat mengakses database server

Mulai dengan menghubungkan ke MySQL sebagai pengguna root MySQL:

$ sudo mysql

Untuk membuat user untuk web server, kita harus menyiapkan ip address web server yang akan kita gunakan.

$ CREATE USER ‘nama_user’@’alamat_ip_webserver’ IDENTIFIED BY ‘password’

Kemudian berikan akun ini akses penuh database:

$ GRANT ALL PRIVILEGES ON *.* TO ‘nama_user’@’alamat_ip_webserver’;

User ini sekarang dapat melakukan operasi apa pun di semua database, dan yang terakhir flush privilegesnya agar MySQL mulai menggunakannya.

$ FLUSH PRIVILEGES;

Step 4. Test koneksi web server ke database server

Masuk ke server web dan sambungkan ke server database menggunakan syntax berikut:

$ mysql –u ‘nama_user’ –h ‘alamat_ip_dbserver’ –p

Jika semuanya bekerja seperti yang diharapkan, kita akan berada di prompt MySQL. Verifikasi koneksi dengan menggunakan perintah:

mysql> mysql> \s

Setelah memverifikasi bahwa Anda dapat terhubung dari web server, keluar dari prompt.

mysql> exit

Menguji Apache Web Server

Membuat file koneksi pada web server

$ sudo nano /var/www/html/koneksidb.php

Lalu masukkan kode berikut:

<?php

// Konfigurasi Database

$host = '192.168.222.2'; // host
$username = 'userweb'; // username database
$password = 'userweb'; // password database
$dbname = 'db_testing'; // nama database

$db = mysqli_connect($host, $username, $password, $dbname);

if ($db) {
echo "Database Terhubung";
} else {
echo "Database Error";
}

Simpan dan tutup dengan menekan CTRL + X, Y, lalu ENTER.

Uji coba di browser.

http://ip_address_webserver/koneksidb.php

Mungkin anda akan menunggu sebentar tapi jika benar akan muncul halaman sukses.

Membuat database wordpress di Database Server

Kembali di Database server kita masuk ke mysql lagi dengan:

$ mysql –u root –p

Dilanjutkan dengan membuat database untuk wordpress dan user-user yang memiliki privileges berbeda dalam database wordpress tersebut.

Lalu flush privileges dan keluar.

mysql> FLUSH PRIVILEGES;

Menginstall Wordpress di Web Server

Pertama, pindah ke direktori /tmp:

$ cd /tmp

Ini mengubah di direktori mana kita berada. Kemudian, masukkan perintah berikut untuk mengunduh WordPress versi terbaru sebagai file terkompresi:

$ wget https://wordpress.org/latest.tar.gz

Ekstrak file terkompresi untuk membuat struktur direktori WordPress:

$ tar xzvf latest.tar.gz

Salin seluruh isi direktori ke root dokumen. Flag -a akan memastikan izin dipertahankan, dan titik di akhir direktori sumber menunjukkan bahwa semua yang ada di dalam direktori harus disalin (termasuk file tersembunyi):

$ sudo cp -a /tmp/wordpress/ /var/www/html

Jalankan perintah di bawah ini untuk mengubah kepemilikan direktori 'wordpress' dan file permission.

$ sudo chown -R www-data:www-data /var/www/html/wordpress/

$ sudo chmod -R 755 /var/www/html/wordpress/

Membuat direkori ‘uploads’ dan ubah izin direktorinya.

$ mkdir /var/www/html/wordpress/wp-content/uploads
$ sudo chown -R www-data:www-data /var/www/html/wordpress/wp-content/uploads

$ sudo cp wp-config-sample.php wp-config.php
$ sudo nano /var/www/html/wordpress/wp-config.php

Untuk mengisi database name, username, password, dan database host. Isi sesuai yang kita gunakan, dimana database name adalah nama database yang kita buat tadi, db user adalah nama user yang dibuat tadi dan db password passwordnya, dan terakhir db host diisi dengan alamat ip database server.


. . .
/** The name of the database for WordPress */
define('DB_NAME', 'database_yang_dibuat_tadi');

/** MySQL database username */
define('DB_USER', 'nama_user');

/** MySQL database password */
define('DB_PASSWORD', 'remote_user_password');

/** MySQL hostname */
define('DB_HOST', 'db_server_ip');
. . .

Setelah mengisi wp-config dengan benar, simpan dan keluar dari file seperti biasa. CTRL + X, Y, dan ENTER. Lalu, arahkan browser ke alamat ip yang terkait dengan web server anda diikuti dengan /wordpress:

http://alamat_ip_webserver/wordpress/wp-admin

Klik Continue untuk melanjutkan setelah memilih Bahasa yang diinginkan namun diajurkan untuk menggunakan English.

Akan diminta informasi untuk wordpress, isi sesuai dengan yang diminta. Jangan khawatir settingan disini bisa diganti nanti.

Lalu install dan log-in.

Ini adalah dashboard wordpress, dari sini website bisa langsung dibuat.

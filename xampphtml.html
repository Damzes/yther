Membuat Website di Wordpress dengan database MySQL

Step 1. Download dan Install aplikasi XAMPP

Langkah pertama dalam cara Anda menginstal WordPress secara lokal adalah dengan mengunduh dan menginstal XAMPP. Installer XAMPP Windows diunduh dari Apache Friends.

https://www.apachefriends.org/index.html

Setelah pengunduhan, jalankan installer seperti biasa dan pilih komponen-komponen yang diperlukan lalu lakukan instalasi. Setelah selesai, luncurkan control panel untuk mulai bekerja dengan XAMPP.

Step 2. Persiapkan modul dan coba server

Untuk menginstal XAMPP dan WordPress dengan benar, diperlukan untuk menjalankan dua modul; Apache danMySQL. Kedua modul dimulai dari panel kontrol XAMPP:

Setelah memulai modul, dapat dilihat statusnya berubah menjadi hijau. Sekarang server lokal dapat diuji berfungsi atau tidak dengan membuka http://localhost/ di browser web.

Jika semuanya berjalan dengan baik, Anda sekarang memiliki server XAMPP yang berfungsi di PC Windows Anda!

Step 3. Persiapan Wordpress

Pertama, download file wordpress dari https://wordpress.org/. Lalu, di Windows, buka folder XAMPP. Kemudian, di folder itu, temukan subfolder bernama htdocs dan buat folder baru untuk webnya. Nama folder ini akan menjadi sub-name yang digunakan untuk mengakses situ. Misalnya jika folder bernama testsite, cara membuka web itu dengan membuka http://localhost/testsite.

Setelah folder dibuat, ekstrak file .zip WordPress yang diunduh dari wordpress.org ke dalamnya.

Step 4. Membuat database untuk WordPress

Klik admin di baris MySQL pada control panel XAMPP.

Halaman phpmyadmin akan terbuka dan dari laman ini, kita dapat membuat database dengan klik Databases yang berada di atas laman lalu mengisi nama database dan klik Create. Ingat nama database karena kan digunakan di proses instalasi Wordpress.

Step 5. Install wordpress

Kunjungi website anda di http://localhost/nama_folder.

Lakukan instalasi sebagaimananya instalasi bekerja sampai diminta Database connection details. Isi database name dengan nama database yang dibuat sebelumnya, nama user diisi dengan root dan kosongkan password.

Submit dan isi lagi yang diminta. Setelah instalasi akan dikembalikan ke login page dan login dengan username password yang dibuat sebelumnya.

Saya sebelumnya sudah pernah membuat website di wordpress, anda bisa membuat sesuatu seperti berikut dengan wordpress.
